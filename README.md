# Mizar64 PRNG
Mizar64 is an fast, minimalist, robust pseudo-random number generator (PRNG) designed for high-performance applications, simulations, and games. 


## Features
Period: $2^{64} - 1$  
State: 64 bits  
Output: 64 bits  
Seed: all 64-bit values ​​except 0
